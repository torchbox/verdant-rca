# Getting Started

* Install [Vagrant](https://wiki.torchbox.com/view/Vagrant)
* ```git clone git@github.com:torchbox/verdant-rca.git```
* ```cd verdant-rca```
* ```vagrant up```
* ```vagrant ssh```
* ```djrun```
* Edit your code locally, browse at [localhost:8000](http://localhost:8000/)
